# Password Manager 1.0

A simple and efficient password manager created in Python, offering both a Graphical User Interface (GUI) and a Command Line Interface (CLI) version.

# Project Structure

- **`main.py`**: The entry point for the Command Line Interface (CLI) version.
- **`gui.py`**: The Graphical User Interface (GUI) of the application, designed for a better user experience.
- **`login.py`**: Contains the application logic, password management functions, and the authentication system.

# How to run the project

1. Clone this repository.
2. Ensure you have Python installed.
3. Run your preferred version:
   - For CLI: `python main.py`
   - For GUI: `python gui.py`
